Web3 VC机构投资项目水平排行榜

作者： Onefly ｜ 单位： 1000X Security ｜ 日期： 2025-09-26
声明： 以下均为个人投研视角，非投资建议。样本来自公开信息与一线社区观察，难免存在偏差，欢迎补充纠错。

机构排名：你不熟，但资源强到离谱的10家

这期不写 Paradigm。特意挑出 低调但回报/资源不输 Paradigm 的三家心头好，再配合七家风格鲜明的机构，聚焦「项目不多、胜率更高」的 Alpha 源头。

评分刻度（主观）

  • 顶级：资源面宽、投后赋能强、赛道判断前瞻、胜率高/赔率高
  • 夯：项目质量与热度显著高于行业均线，上所与生态资源链接强
  • 人上人：稳定输出，选赛道不跟风，有独到洞察
  • NPC：存在感一般或赔率/胜率不匹配，需择优跟踪

01 Founders Fund（@foundersfund）— 今年我心中最强之一 🔥

画像： 顶级美元系顶层资源，筛选标准严苛；项目一旦入选，上游资本、CZ/大所、流量/KOL 全链路协同明显。
优势：

  • 选赛道不拥挤，To-C 热点与 To-B 基础设施兼顾
  • 投后资源“推得动”——上交易所、上生态、上叙事
  • 领投项目热度溢价显著

代表性项目（节选）
Avantis / Plasma / Caldera / ABS / Avail / Lagrange / Maverick / Ondo / Starknet / OpenSea

评级： 夯（接近顶级）
参与策略： 追踪其早期领投 → 观察所方互动与生态 Grants → 二级择时看资源兑现速度。

02 CoinFund（@coinfund_io）— 低调的高胜率猎手 🎯

画像： 样本量不大但质量极高；兼顾自孵与外部投资。
优势：

  • 赛道差异化明显，常常避开拥堵主线
  • 前中期深度研究与组合风控到位
  • 多个尚未发币的高质量储备项目

代表性项目（节选）
Giza / Ether.fi / Gunzilla / Manta / Neutron / LayerZero / Hyperlane / Polygon / World / Syndicate / Ondo / Biconomy / Moonbeam / ParaSwap / Balancer / NuCypher / CoinList

评级： 夯
参与策略： 跟踪其研究报告/投后文章 → 提前埋伏代币化前的生态互动与空投路径。

03 Galaxy（@galaxyhq）— CoinList 常青树背后的“高估值范本” 🧭

画像： 上轮周期里顶级项目常见投资人；对一级 & 二级联动敏感。
优势：

  • 早期优质供给筛选强（尤其 L1/L2、基础设施）
  • 与CoinList 系节奏关联度高（历史观察）
  • 高估值强资源打法的典型代表

代表性项目（节选）
Sonic / Usual / Plume / Avalanche / Renzo / Babylon / Avantis / Boundless / Celestia / Hyperlane / Polygon / Luna / 1inch / Immutable / NuCypher

评级： 夯
参与策略： 借其投后清单做「空投/节点/测试网」埋伏；二级阶段关注流动性做市与所方节奏。

04 Delphi Digital（@Delphi_Digital）— 研报天花板 + 孵化器 🧪

画像： 既是最强研究院，也是能把研究变成项目的孵化器。
优势：

  • 宏观到赛道到项目的全栈框架
  • 孵化 + 投顾，投后赋能极强
  • 报告对市场风向具备解释力

代表性项目（节选）
0G Labs / Grass / io.net / Initia / Pixelmon / Pyth / Boundless / Ethena / Neutron / Anoma / Celestia / Sei / Jito / Cyber / Beta / LayerZero / dYdX / Lido / Syndicate / Illuvium

评级： 顶级
参与策略： 订阅其研究脉络 → 关注孵化线（从工具到生态入口）→ 叙事前置卡位。

05 Hashed（@hashed_official）— 韩系代表，游戏倾向明显 🎮

画像： 上轮牛市接近顶级；近两年GameFi 权重上升，但兑现不如预期。
优势/痛点：

  • 亚洲资源与本地生态人脉面广
  • 项目参差，游戏方向赔率波动大

代表性项目（节选）
b³ / Sonic / Humanity / Saros / Story / NEAR / Ape / Aptos / Loka / Cyber / World / Axie / dYdX / Injective / zkSync / The Sandbox / Luna / NuCypher

评级： 人上人（中上游）
参与策略： 精选赛道（链上游戏基础设施 > 纯游戏代币），关注运营与发行合作落地速度。

06 GSR（@GSR_io）— 一线做市商，投资参考度一般 ⚖️

画像： 与多家交易所保持良好关系；流动性与报价能力强。
优势/痛点：

  • 做市赋能真能力
  • 投资样本分散、参考信号噪声大

代表性项目（节选）
Usual / Kernel / Vana / MYX / Nillion / Ethena / Parti / Sei / Morpho / Saga / Beta / dYdX

评级： 人上人（偏做市能力）
参与策略： 作为流动性侧信号，结合所方/做市安排观察资金面。

07 Framework Ventures（@hiFramework）— 顶级调研 + 孵化能力 🧠

画像： 以深度研究与投后建设闻名；近期因 Plasma 孵化刷屏。
优势：

  • 强主题研究 → 项目孵化的闭环
  • 与DeFi/基础设施高度耦合，中长期复利

代表性项目（节选）
Plasma / Towns / Bear / Pixels / Babylon / Immutable / Illuvium / Jito / DODO / Aave

评级： 顶级
参与策略： 跟随其研究主题，在孵化线早期参与生态交互/节点/测试。

08 BITKRAFT（@BITKRAFTVC）— 游戏生态重仓，赔率不稳 🎲

画像： 传统游戏与链游的交叉资本；兑现高度依赖大作落地。
优势/痛点：

  • 资源在内容/工作室/发行
  • 多个项目发币之前即折损，赔率承压

代表性项目（节选）
Sonic / Immutable / YGG

评级： NPC（受赛道周期影响大）
参与策略： 只跟确定性极强的基础设施与发行渠道级资产。

09 Dao5（@daofive）— “青蛙”头像下的硬实力 🐸

画像： 形象接地气，但项目选择与结构化能力不俗。
优势：

  • AI×Crypto、再质押、数据要素等新兴交叉方向把握到位
  • 组合里进顶级气质稍弱，但胜率可观

代表性项目（节选）
Stakestone / Sahara / Bard / Move / 0G / BounceBit / Story / Bear / EigenLayer

评级： 人上人
参与策略： 重点跟AI/再质押/数据层三条主线，做生态级联动套利。

10 Mechanism Capital（@MechanismCap）— 中小盘百倍猎手 🧬

画像： 上轮在打新平台/中小盘 DeFi里频出佳绩；本轮参与感偏弱。
优势：

  • 敢投高赔率中小盘
  • 价值捕捉阶段更偏叙事爆发前

代表性项目（节选）
Bio / Home / Xion / Pixels / Humanity / MON / Morpho / NEAR / Platypus / Genopets / DeHorizon / Merit Circle / Highstreet / Biconomy / Pendle / Benqi

评级： NPC（若回归主动，将上调）
参与策略： 追踪其早期小众下注；结合链上活跃/新资金进入做事件驱动。

这一期的三个“最喜欢”

  • Founders Fund：资源与筛选同层顶配，项目热度与所方资源双兑现
  • CoinFund：低频高质的范本；自孵 + 投研闭环
  • Framework：研究 → 孵化 → 生态的长期复利王道

小结

  • 本期挑选的是投得不多但胜率/回报高的机构。
  • 许多尚未发币的储备项目未列入代表清单，但机构路线已经给出了足够强的前导信号。
  • 你的排名 ≠ 我的排名。维度不同，结论就不同。关键是：把机构的“资源与方法”转化为自己的“可执行策略”。
