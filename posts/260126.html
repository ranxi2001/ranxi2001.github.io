Clawdbot：你的永不休息的 AI 助手 - 完整部署与使用指南

最近 AI 圈被一个名为 Clawdbot 的开源项目刷屏了，它不仅带火了 Mac Mini 的销量，甚至导致 VPS 资源紧缺。这个被称为”永不休息的 AI 员工”的工具到底有何魔力？本文将基于社区精华内容，为你详细解析 Clawdbot 的核心能力、部署方案及实战应用。

Clawdbot 概述

Clawdbot（也称 Clawbot 或 ClawdBot）是由开发者 Peter Steinberger 和社区维护的开源个人 AI 助手项目。它的核心理念是创建一个本地优先、去中心化的 AI 代理，能在用户自己的硬件上运行，通过即时通信平台进行交互。吉祥物是一只龙虾（🦞），象征其”抓取”任务的能力。

关键优势

本地运行，隐私至上

  • 数据不上传云端，完全在本地处理
  • 支持 macOS、Linux 和 Windows（通过 WSL2）
  • 推荐在专用设备上运行以隔离环境

多渠道无缝接入

  • 支持 WhatsApp、Discord、Telegram、X（Twitter）、Gmail
  • 兼容 Signal、iMessage、Microsoft Teams 等
  • 一次部署，多平台使用

实际执行任务，而非仅限聊天

  • 浏览网页、填写表单、读写文件
  • 执行 Shell 命令、运行脚本
  • 管理邮箱、日程安排
  • 自动化操作电脑、浏览器等

自我进化能力

  • AI 可生成新技能（MCP - Modular Capability Packs）
  • 支持浏览器控制、Canvas 画布、定时任务
  • 内置 Exa MCP（免费网页搜索）、YouTube Clipper 等技能

高性能模型支持

  • 推荐使用 Anthropic 的 Claude Opus 4.5 模型
  • 结合 MiniMax M2.1 等模型实现高性能、低成本
  • 需要 Anthropic Pro/Max 订阅

适用场景

作为”永不休息的 AI 员工”，用户可以：

  • 躺在床上通过 Telegram 重建网站
  • 监控代码仓库，自动提交修复
  • 自动回复邮件、安排会议
  • 值机登机、抓取数据
  • 远程操控电脑执行复杂任务

用户 @techfrenAJ 分享：”5 分钟部署，躺在床上看 Netflix 时重建网站。” 这正是 Clawdbot 的魅力所在。

部署选项详解

Clawdbot 支持多种部署方式，满足不同用户的需求和预算。

方案一：AWS 免费套餐（推荐新手，零成本）

基于 @op7418 整理的视频教程，这是最适合新手的部署方式。

部署步骤

  1. 启动 EC2 实例

    • 在 AWS 控制台搜索 EC2，点击橙色的”Launch Instance”按钮
    • 实例命名随意，系统选择 Ubuntu
    • 实例类型搜索”free”，选最大的免费档（8GB 内存的 t4g.medium 或类似）
    • 不需创建其他资源，确保在免费套餐范围内

  2. 连接实例

    • 实例创建后，进入 Instance ID
    • 点击”Connect”（浏览器内打开 SSH 终端）

  3. 一行命令安装

    • 访问 clawd.bot 官网
    • 复制安装命令：
      curl -sSfL https://install.clawd.bot | bash
    • 在终端粘贴执行，安装过程需几分钟

  4. 配置 Clawdbot

    • 风险声明：同意文件系统访问、邮件/密钥权限
    • Quick Start：使用 Anthropic Max 订阅
    • 模型选择：运行本地命令获取 Token，选 Opus 4.5 模型
    • 关联渠道：如 Telegram，生成配对码，输入实例
    • 配置技能：Homebrew、Node 包管理器（可跳过）
    • 启用功能：内存、钩子等

  5. Hatch（孵化）- 定义 AI 身份

    • 示例："你是 Jinx，我是 AJ。AJ 是内容创作者和软件工程师。"
    • 设置时区、交互氛围（如 Romantic）

  6. 激活技能

    • 安装 Exa MCP（免费网页搜索技能）
    • 命令：”创建一个技能，封装这个 MCP”

特色功能

  • Meta Ray-Ban 眼镜集成：实时价格比较
  • 整个过程 < 2 分钟设置完成
  • 免费套餐内完全免费（需监控使用量）

安全提示：别在 AWS 实例上登录你的常用账号，避免风险。

方案二：Mac Mini（热门选择，稳定本地部署）

Mac Mini 成为 Clawdbot 最受欢迎的部署平台，甚至带火了 M4 芯片版的销量。

为什么选择 Mac Mini？

  • 价格亲民：国补 + 教育优惠约 3100 元
  • 兼容性好：macOS 原生支持，无需额外配置
  • 功率低：安静、省电，适合 7/24 运行
  • 占地小：许多用户用吃灰的旧 Mac Mini 复活

部署方式

  • 直接在 Mac Mini 上运行安装命令
  • 支持隔离环境，安全性高
  • 用户反馈：通过 Telegram 监控 Claude 编码、自动拉取代码仓库、打开 VS Code、运行测试、生成修复、提交代码

替代方案

  • 二手电脑、树莓派、旧笔记本
  • 但 Mac Mini 运行更稳定，兼容性最佳

用户 @nash_su 评价：”Mac Mini M4 被下单爆火，终于让吃灰设备发光发热。”

方案三：VPS 或云服务器（云端在线，灵活扩展）

优势

  • 永远在线，无需本地设备
  • 成本低（5 分钱/天算力）
  • 可扩展节点，支持多实例

推荐服务

  • 懒猫微服（@manateelazycat 推荐）：
    • 自带内网穿透、Linux Docker
    • 3000+ 应用 MCP
    • AI 算力舱（275T 真算力、无限 Tokens）
    • 库存多，优惠力度大

部署步骤

  • 类似 AWS，在 VPS 上运行安装命令
  • 支持虚拟机测试（隔离风险）

注意事项

  • 资源紧缺（硅相关一切短缺）
  • 国内 Telegram 门槛高，可能影响使用

安装与配置细节

模型配置

API Key 获取

  • 需要 Anthropic API Key
  • Claude Code OAuth 限制外部使用，需单独申请

推荐模型组合

  • Claude Opus 4.5：高性能任务（需 Pro/Max 订阅）
  • MiniMax M2.1：@MiniMax_AI 推荐
    • Trader Joes 质量
    • Amazon Prime 速度
    • Temu 价格

渠道配对

  1. 生成配对码
  2. 在聊天工具中输入配对码
  3. 支持多渠道同步

技能扩展

内置技能

  • Exa MCP：免费网页搜索/深度搜索，无需 API 密钥
  • YouTube Clipper：剪辑视频、翻译字幕、生成文案

自定义技能

  • AI 可自我生成技能
  • 示例：”创建一个技能，封装这个 MCP”

Web UI

  • 安装后生成本地界面
  • 与终端同步，直观管理

时间成本

  • 设置时间：< 5 分钟（一行命令）
  • 安装时间：需耐心等待几分钟

风险与注意事项

安全风险

权限极高

  • 文件访问、Shell 执行、账号操作
  • 易受提示词注入攻击
  • 可能导致密码泄露、文件误删
  • RPA 行为可能导致账号封禁

推荐做法

  • 虚拟机/不重要账号测试
  • 仔细阅读并同意风险声明
  • 不登录常用账号
  • 隔离环境运行

用户反馈

  • 部分用户因权限太大已卸载
  • 大多数用户认为隔离环境下可接受
  • 需要理性评估风险

其他限制

国内使用门槛

  • Telegram 门槛高，可能限制普及
  • 需要稳定的网络环境

理性评估

  • 开源无大厂背书，安全性待长期考验

常见问题 (FAQ)

Q: Clawdbot 完全免费吗？
A: Clawdbot 本身是开源免费的，但需要：

  • Anthropic API Key（需订阅 Pro/Max）
  • 运行环境（AWS 免费套餐、Mac Mini 或 VPS）
  • 可选的 MCP 技能（部分免费，如 Exa MCP）

Q: 安全吗？会泄露隐私吗？
A: Clawdbot 是本地优先设计，数据不上传云端。但因权限极高，建议：

  • 在隔离环境测试
  • 不使用常用账号
  • 仔细评估风险

Q: 哪种部署方式最适合我？
A:

  • 新手/低成本：AWS 免费套餐
  • 稳定/本地：Mac Mini（或旧电脑）
  • 云端/灵活：VPS 或懒猫微服

Q: 需要什么技术背景？
A: 基础的命令行操作即可，安装是一行命令。配置过程有引导，跟着步骤走即可。

Q: 支持中文吗？
A: 支持，Claude Opus 4.5 模型有很好的中文能力。社区也有中文用户群。

Q: 和 Claude Code 有什么区别？
A:

  • Clawdbot：本地代理，多渠道接入，7/24 在线
  • Claude Code：官方工具，更聚焦开发场景
  • 根据需求选择，两者可互补

结语

Clawdbot 代表了个人 AI 助手的新趋势：本地优先、去中心化、实际执行任务。它不仅仅是聊天机器人，而是真正的”AI 员工”，能帮你处理数字世界中的琐事。

无论你选择 AWS 免费套餐、Mac Mini 还是 VPS，部署过程都非常简单（< 5 分钟）。但在享受便利的同时，也要理性评估安全风险，做好隔离措施。

如果你正在寻找一个能真正帮你自动化任务的 AI 助手，Clawdbot 值得一试。它可能不会让你的 Mac Mini 吃灰了，反而会让它成为你最得力的数字助手。
