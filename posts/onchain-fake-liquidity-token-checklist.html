链上行为学：如何判断一个币是不是“假流动性”

很多新币刚上线时，Dexscreener 上的 Liquidity 数字看起来很好看：几十万、几百万美元，好像池子很厚，买卖都不会出问题。

但这个数字只能作为参考，不能直接等同于“真实可成交深度”。

尤其是在 Meme 币、新项目首发、低市值高波动标的里，流动性看起来很厚，真正卖出时却发现滑点爆炸，是非常常见的坑。

这篇整理自 Mojito 的 X 线程，我会把原线程扩展成一套更适合实战的检查框架：买一个新币前，如何判断它是不是“假流动性”。

先定义：什么是假流动性？

所谓“假流动性”，不是说池子里完全没有钱，而是指：

表面指标显示有流动性，但当前价格附近真实可成交深度很差，或者流动性被少数主体控制，随时可能撤走。

它常见于几种情况：

  • V3 池总流动性很高，但资金集中在远离当前价格的区间；
  • 多个池子分散显示 liquidity，但主交易池深度很差；
  • 小额交易正常，大额卖出滑点极高；
  • LP token 或 V3 LP NFT 主要由项目方/少数钱包控制；
  • 成交量看起来很高，但只是少数钱包互刷；
  • 上涨过程中项目方持续撤 LP，慢慢抽走买盘承接。

一个简单公式是：

flowchart LR
    A[表面 Liquidity] --> B{是否真实可卖出}
    B -->|滑点低| C[相对健康的流动性]
    B -->|滑点高| D[可能是假流动性]
    D --> E[继续查 V3 区间]
    D --> F[继续查 LP 控制权]
    D --> G[继续查成交量与钱包分布]

真正重要的不是“池子显示有多少钱”，而是：你要卖的时候，有没有足够买盘接住你。

一、不要把 Liquidity 数字当安全垫

很多人看新币，只看 Dexscreener 上的两个数字：

  • Liquidity；
  • 24h Volume。

如果 Liquidity 显示几十万，就觉得至少不会一卖归零。

这个判断太粗糙。

因为 Dexscreener 显示的流动性，通常只是池子层面的表面统计。它不能直接回答几个更关键的问题：

  • 当前价格附近有多少可成交流动性？
  • 你卖 1 ETH、5 ETH、10 ETH 时滑点分别是多少？
  • LP 是锁住的，还是项目方随时可以撤？
  • 成交量来自真实用户，还是少数钱包互刷？

所以，Liquidity 只能当入口，不能当结论。

判断一个币能不能碰，至少要从“池子结构、卖出滑点、LP 控制权、成交量质量”四个方向一起看。

二、V2 和 V3 的流动性不是一回事

很多人看池子时不区分 V2 和 V3，这是第一个大坑。

V2：池子深度相对直观

Uniswap V2 这类 AMM 池比较好理解。

池子里有多少 token 和 ETH，大致就能判断当前价格附近的交易深度。虽然大额交易同样会滑点，但至少流动性分布相对连续。

V3：总流动性不等于当前价格深度

V3 完全不一样。

V3 是区间流动性。LP 可以只把资金放在某一个价格区间里，而不是覆盖所有价格。

这会带来一个问题：

一个 V3 池显示 Liquidity 很高，但如果这些资金不在当前价格附近，你实际交易时仍然会遇到很差的深度。

可以把 V2/V3 的区别理解成这样：

flowchart TD
    A[同样显示高 Liquidity] --> B[V2 池]
    A --> C[V3 池]
    B --> D[资金通常覆盖整条价格曲线]
    C --> E[资金可能集中在某些价格区间]
    D --> F[表面流动性更接近真实深度]
    E --> G[必须检查当前价格附近深度]

所以，看到 V3 池时，不要只看总 Liquidity，而要追问：这些流动性是不是刚好在当前价格附近？

三、最直接的方法：模拟卖出，看真实滑点

判断一个币有没有真实流动性，最直接的方法不是看别人怎么说，而是自己模拟卖出。

很多人买币前只模拟买入，很少模拟卖出。但你真正需要知道的是：

  • 如果我买进去，能不能顺利卖出来？
  • 卖出 1 倍仓位、2 倍仓位时，滑点会不会突然爆炸？
  • 小额能卖，大额是不是完全卖不动？

实操步骤

  1. 打开 Dexscreener，找到这个币的主池；
  2. 点击 Trade，跳转到对应 DEX；
  3. 把交易方向切换为：卖出该代币，换成 ETH/USDC；
  4. 假设自己持有一定数量，分别模拟小额、中额、大额卖出；
  5. 观察报价、Price Impact、Minimum received。

不要只测一个金额。建议至少测三档：

  • 小额：确认是否能正常交易；
  • 中额：接近自己计划买入金额；
  • 大额：模拟极端情况下能不能退出。
flowchart LR
    A[准备买入新币] --> B[先模拟卖出]
    B --> C[小额卖出]
    B --> D[计划仓位卖出]
    B --> E[更大金额卖出]
    C --> F{滑点是否稳定}
    D --> F
    E --> F
    F -->|稳定| G[流动性相对可信]
    F -->|大额爆炸| H[警惕假流动性]

这里有个实战判断：

小额卖出正常，不代表池子健康；计划仓位卖出也能接受，才有参考价值。

四、流动性不只看多少，还要看是谁加的

池子里有钱是一回事，钱由谁控制是另一回事。

一个池子的 LP 如果主要由项目方或少数钱包控制，就要继续查。因为这意味着对方可能拥有撤走流动性的能力。

重点看几个问题：

  • LP token 在谁手里？
  • LP 有没有锁？
  • 锁仓时间到什么时候？
  • 是否频繁加 LP / 撤 LP？
  • V3 的 LP NFT 有没有被转走？
  • LP 钱包和项目方、部署者、早期持仓地址有没有关联？

很多项目不会一开盘就直接砸盘，而是采用更隐蔽的方式：

  1. 先加一个看起来不错的池子；
  2. 拉盘制造赚钱效应；
  3. 吸引散户买入；
  4. 随着买盘进入，逐步撤出流动性；
  5. 最后留下高位筹码和越来越差的退出深度。

这个过程不一定表现为一根暴跌线，但本质仍然是在抽走市场承接。

五、成交量和流动性要匹配

还有一种情况也很常见：

Liquidity 很低，但 Volume 很高。

表面看，这个币 24h 成交量很大，好像很活跃。

但如果继续查，可能会发现成交量只是少数钱包反复买卖刷出来的。

这时候要看三个匹配关系。

1. Volume 增长时，Holder 是否同步增长

如果成交量很高，但持有人增长很慢，说明交易活跃度可能集中在少数地址之间。

2. 买卖是否集中在少数钱包

如果大部分交易来自几个固定钱包，尤其是反复买卖、金额相似、时间间隔规律，就要警惕刷量。

3. 价格上涨是否伴随真实买盘扩散

健康上涨通常会伴随更多地址参与，而不是只有几个钱包来回对倒。

flowchart TD
    A[Volume 很高] --> B{Holder 是否同步增长}
    B -->|是| C[继续观察买盘质量]
    B -->|否| D[可能刷量]
    C --> E{交易是否分散}
    E -->|分散| F[活跃度相对真实]
    E -->|集中少数钱包| G[警惕对倒和假活跃]
    D --> G

成交量不是越高越好，关键是成交量背后有没有真实用户和真实承接。

六、判断假流动性的信号清单

遇到下面这些情况，要提高警惕：

  • Liquidity 看起来很高，但主池成交量很低；
  • 多个池子分散流动性，真实成交深度很差；
  • V3 总流动性高，但当前价格附近没深度；
  • 小额卖出正常，大额卖出滑点爆炸；
  • LP 由少数钱包控制；
  • LP 没有锁，或者锁仓时间很短；
  • 上涨过程中 LP 被频繁撤出；
  • Volume 很高，但 Holder 增长不明显；
  • 买卖集中在少数钱包，且交易节奏异常规律；
  • 社群持续强调“池子很厚”，但回避 LP 控制权和滑点问题。

这些信号不一定单独构成死刑，但如果同时出现两三个，就应该明显降低仓位，甚至直接放弃。

七、买新币前的 5 分钟检查流程

如果只给你一个最小可执行流程，我建议这样做：

Step 1：看主池

确认主池在哪里，是 V2 还是 V3，流动性是否集中在一个主要池子里。

Step 2：模拟卖出

按计划仓位模拟卖出，重点看 Price Impact 和 Minimum received。

Step 3：查 LP 控制权

看 LP token 或 V3 LP NFT 在谁手里，有没有锁，锁到什么时候。

Step 4：看成交分布

看成交量是否由真实用户贡献，还是少数钱包反复对倒。

Step 5：看动态变化

观察上涨过程中 LP 有没有持续撤出，主池深度有没有变差。

flowchart TD
    A[准备买入] --> B[确认主池 V2 / V3]
    B --> C[模拟卖出看滑点]
    C --> D[检查 LP 控制权与锁仓]
    D --> E[核对 Volume 与 Holder]
    E --> F[观察 LP 动态变化]
    F --> G{风险是否可接受}
    G -->|是| H[小仓位参与]
    G -->|否| I[放弃或等待]

这套流程不保证你一定赚钱，但能过滤掉很多“看起来热闹、实际无法退出”的坑。

结语：真正的流动性，是你能退出的流动性

链上交易里，最容易骗人的指标往往不是价格，而是“看起来很厚”的池子。

因为价格可以拉，成交量可以刷，Liquidity 可以堆到不关键的位置，LP 也可以在买盘进来后慢慢撤。

所以判断新币安全性时，不要只问：

这个币池子有多厚？

更应该问：

如果我现在买进去，按我的仓位卖出时，市场能不能接住我？

真正的流动性，不是页面上显示的那个数字，而是你需要退出时，仍然存在的真实承接。
