我选的服务器和 API 可能不是最便宜、性价比最高的，但是我认为是最简单的，适合 0 技术基础的同学学习搭建。我自己也是刚自学两天，不足之处请指教。

第一步：购买腾讯云轻量服务器

点击购买链接

费用约 40 元一个月，买完先放着。

第二步：创建 Telegram Bot

在 Telegram 中搜索 BotFather（注意有蓝标认证），并点击进入。

发送指令：

/newbot

然后按照提示：

  1. 取个名字（Name）。
  2. 取个用户名（Username，必须以 bot 结尾）。

BotFather 发给你的这一长串字符（Token）就是你的 API 密钥，请保存好，不要泄露。

示例 Token：
8232094405:AAFTXBMwkhRfE08bmqjOw0b27N-vOjlHbGg

第三步：购买 OpenClaw API

我是在闲鱼买的中转 API（每个 API 都需要中转，所以用法不一样）。如果你在其他商家购买，请咨询商家如何使用。

【闲鱼】HU071 「快来捡漏【OpenClaw 一键配置 API Claude Opus 4.5 Opus-4-6】」

点击链接购买后会收到一串密钥，例如：
H58JA72B-PNHV-XXXX-OGIH2

现在所有准备工作就绪，开始部署。

第四步：部署 OpenClaw

回到腾讯云控制台，找到刚才购买的机器，点击”登录”按钮，在弹窗中再点击”登录”，进入 Web 终端。

在终端中输入以下命令并回车：

clawdbot onboard

之后使用键盘的 上下左右方向键 和 回车键 来操作界面。

初始化配置

看到提示 I understand this is powerful and inherently risky. Continue?
按方向键 ← 选择 Yes，回车。

选择 QuickStart，回车。

选择 Use existing values，回车。

选择 Anthropic，回车。

选择 Anthropic API key，回车。
把刚才闲鱼买的密钥（如 H58JA72B-PNHV-XXXX-OGIH2）粘贴进去，回车。

选择模型（推荐 Claude Sonnet 4.5，4.6 很贵），回车。

配置 Telegram 频道

接下来选择你的机器人聊天频道。如果你想部署到其他频道（如 QQ、微信、Discord、飞书），可以参考腾讯云教程。

这里我们选择 Telegram (tg)，回车。

输入你从 BotFather 获取的 Bot Token（如 8232094405:AAFT...），粘贴并回车。

提示 Configure skills now? (recommended)
这里选择 No。

选择 Restart。

选择 Do this later。

之后会出现一行红字提示，此时把这个网页关掉。

绑定验证

回到腾讯云终端，再次登录。

回到 Telegram，找到你刚创建的 Bot，给它发一条任意消息。

Bot 会回复一条验证码消息，格式如：
openclaw pairing approve telegram XXXX

把你收到的那条验证命令粘贴进终端，回车。例如：

openclaw pairing approve telegram CLUPKDAE

配置 API Key

现在 Telegram 配置好了，最后配置 API Key。

使用以下命令（记得替换 你的Key）：

OPENCLAW_CLAUDE_KEY="你的Key" npx openclawapi@latest preset-claude

例如：

OPENCLAW_CLAUDE_KEY="H58JA72B-PNHV-XXXX-OGIH2" npx openclawapi@latest preset-claude

  1. 键盘输入 y，回车。
  2. 提示保留配置，直接回车。
  3. 选择模型：选择 Claude Sonnet 4.5，回车。
  4. 设为默认模型？输入 Y，回车。
  5. 是否立即通过 OpenClaw Gateway 测试？输入 Y，回车。

如果通过测试，说明 Bot 已经搭建好了，给它发消息验证即可。

故障排查

如果没有通过测试，或者 Bot 没反应，尝试执行这两条命令重启服务：

openclaw gateway stop
openclaw gateway start

如果后期更新了 API Key，也需要执行这两条命令重启 Bot。
